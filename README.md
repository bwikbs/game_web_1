# ⛏ 웹크래프트 (WebCraft)

Three.js로 만든 마인크래프트 스타일 복셀 웹게임. 외부 에셋 없이 텍스처까지 전부 코드로 생성합니다.

## 실행

```bash
npm install
npm run dev     # http://localhost:5173
```

## 조작

| 키 | 동작 |
|---|---|
| W A S D | 이동 |
| Space | 점프 / 수영 |
| Shift | 달리기 |
| 좌클릭 | 블록 캐기 |
| 우클릭 | 블록 설치 |
| 1~9 / 휠 | 핫바 블록 선택 |
| ESC | 일시정지 |

## 구조

- `src/noise.js` — 시드 기반 Perlin 노이즈 + fBm (지형 생성용)
- `src/blocks.js` — 블록 정의 + 캔버스로 절차 생성하는 16px 텍스처 아틀라스
- `src/world.js` — 청크(16×16×64) 데이터 생성·메싱(노출면만 + AO), 동적 로딩, 복셀 DDA 레이캐스트
- `src/player.js` — 1인칭 조작, AABB 충돌 물리(중력/점프/수영), 안전 스폰 탐색
- `src/main.js` — 게임 루프, 핫바/HUD, 블록 캐기·설치, 포인터락

## 특징

- 절차적 무한 지형: 언덕, 산(눈), 해변, 호수, 나무
- 플레이어 주변 반경 5청크 동적 로드/언로드 (수정 내용은 세션 내 보존)
- 청크 메싱: 노출된 면만 생성 + 꼭짓점 앰비언트 오클루전
