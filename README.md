# ⛏ 웹크래프트 (WebCraft)

Three.js로 만든 마인크래프트 스타일 복셀 웹게임. 외부 에셋 없이 텍스처까지 전부 코드로 생성합니다.

## 실행

```bash
npm install
npm run dev     # http://localhost:5173
```

### 멀티플레이 (선택)

```bash
npm run server  # ws://localhost:8081 릴레이 서버
```

서버가 켜져 있으면 클라이언트가 자동 접속해 블록 변경과 플레이어 위치를 실시간 동기화합니다
(다른 호스트는 `?ws=ws://주소:8081` 쿼리로 지정). 서버가 없으면 자동으로 싱글 플레이로 동작합니다.

## 조작

| 키 | 동작 |
|---|---|
| W A S D | 이동 |
| Space | 점프 / 수영 |
| Shift | 달리기 |
| 좌클릭 | 블록 캐기 |
| 우클릭 | 블록 설치 |
| 1~9 / 휠 | 핫바 블록 선택 |
| ESC | 일시정지 |

## 구조

- `src/noise.js` — 시드 기반 2D/3D Perlin 노이즈 + fBm
- `src/blocks.js` — 블록 정의 + 캔버스로 절차 생성하는 16px 텍스처 아틀라스
- `src/world.js` — 청크(16×16×64) 데이터 생성·메싱(노출면만 + AO), 동굴 굴착, 동적 로딩, 복셀 DDA 레이캐스트
- `src/player.js` — 1인칭 조작, AABB 충돌 물리(중력/점프/수영), 안전 스폰 탐색
- `src/sky.js` — 낮·밤 주기 (해/달, 하늘·안개·지형 밝기)
- `src/audio.js` — WebAudio 절차 생성 효과음 (캐기/설치/걸음/점프/입수)
- `src/storage.js` — localStorage 자동 저장/불러오기
- `src/net.js` / `server.js` — WebSocket 멀티플레이 (블록·위치 동기화)
- `src/main.js` — 게임 루프, 핫바/HUD, 블록 캐기·설치, 포인터락

## 특징

- 절차적 무한 지형: 언덕, 산(눈), 해변, 호수, 나무, 3D 노이즈 동굴
- 240초 낮·밤 주기 (해/달 궤도, 하늘·안개 색 보간)
- 플레이어 주변 반경 5청크 동적 로드/언로드
- 청크 메싱: 노출된 면만 생성 + 꼭짓점 앰비언트 오클루전
- 월드 수정·위치·시간 자동 저장 (5초마다 + 종료 시)
- 외부 에셋 0개: 텍스처는 캔버스로, 사운드는 WebAudio로 합성
